# gaja-awesean

해커/터미널 테마의 개인 앱 런처 허브. 터미널 스타일 UI로 개인 프로젝트들을 한눈에 모아볼 수 있습니다.

## 포함된 앱

| 앱 | 설명 | 기술 스택 | 링크 |
|---|---|---|---|
| **MovieTalk** | YouTube 영어 발음 학습 | React, Vite, Claude API | [앱](https://movietalk-psi.vercel.app) · [GitHub](https://github.com/nothinking/movietalk) |
| **YT Organizer** | YouTube 구독 채널 카테고리 정리 | Next.js, Supabase, TypeScript | [앱](https://owesean-yt-organizer.vercel.app) · [GitHub](https://github.com/nothinking/owesean-yt-organizer) |
| **Jeju Weather** | 제주도 실시간 날씨·CCTV 대시보드 | Vanilla JS, CCTV, Windy | [앱](https://nothinking.github.io/awesean-jeju-weather/) · [GitHub](https://github.com/nothinking/awesean-jeju-weather) |
| **Hit the Coffee** | AI 메뉴 인식 단체 커피 주문 | Next.js 15, AI OCR, Supabase | [앱](https://hit-the-coffee.vercel.app) · [GitHub](https://github.com/nothinking/hit-the-coffee) |

## 기술 스택

- Vanilla HTML / CSS / JS (빌드 도구 없음)
- [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) 폰트
- CRT 스캔라인, 비네팅, 타이핑 애니메이션 등 터미널 효과
- 반응형 2열 그리드 레이아웃

## 로컬 실행

별도 빌드 과정 없이 `index.html`을 브라우저에서 열면 됩니다.

```bash
# 간단한 로컬 서버 실행
npx serve .
# 또는
python3 -m http.server 8000
```
